## 📋 Descripción del Proyecto

FanKit es una tienda online profesional de camisetas de fútbol desarrollada con tecnologías modernas. El proyecto incluye un frontend completamente funcional y está preparado para la implementación del backend.

## 🚀 Características Principales

- ✅ **Frontend Completo**: 8 páginas completamente funcionales
- ✅ **Responsive Design**: Optimizado para todos los dispositivos
- ✅ **Sistema de Autenticación**: Login, registro y recuperación de contraseña
- ✅ **Carrito de Compras**: Funcionalidad completa con persistencia
- ✅ **Galería de Productos**: Con zoom y navegación avanzada
- ✅ **Sistema de Búsqueda**: Filtrado en tiempo real
- ✅ **Gestión de Stock**: Contador en tiempo real
- ✅ **Efectos Visuales**: Animaciones y transiciones profesionales

## 🚀 Instalación y Uso

### Prerrequisitos
- Navegador web moderno
- Servidor web local (Live Server, XAMPP, etc.)

### Instalación
1. Clona o descarga el proyecto
2. Abre la carpeta `frontend` en tu editor
3. Inicia un servidor web local
4. Navega a `index.html`

### Desarrollo
```bash
# Para desarrollo frontend
cd frontend
# Usar Live Server o similar para servir archivos

# Para desarrollo backend (próximamente)
cd backend
npm install
npm run dev
```
