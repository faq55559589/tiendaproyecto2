  "Camisetas Retro" apunta a "#" en lugar de "retros.html"

6. MEJORA: Badge contador en el carrito
-----------------------------------------------------------
¿QUÉ ES UN BADGE?
  Es una pequeña etiqueta visual (generalmente un círculo con número)
  que muestra información importante de forma destacada.

ANTES: Carrito (0)
AHORA: Carrito [0] <- El número aparece en un círculo azul

VENTAJAS:
  - Más visible
  - Estilo moderno (usado por Amazon, MercadoLibre, etc.)
  - Fácil de actualizar con JavaScript más adelante

CÓDIGO:
  <span class="badge bg-primary">0</span>

CLASES BOOTSTRAP USADAS:
  - badge: Estilo de insignia
  - bg-primary: Color de fondo azul (color principal del tema)

-----------------------------------------------------------
7. MEJORA: Efectos hover en tarjetas de productos
-----------------------------------------------------------
¿QUÉ ES HOVER?
  Es el efecto que ocurre cuando pasas el mouse por encima de un elemento.

EFECTOS AGREGADOS:
  1. Sombra más pronunciada (elevación)
  2. Transformación suave (se eleva ligeramente)
  3. Transición animada

CSS AGREGADO:
  .card {
    transition: transform 0.3s ease, box-shadow 0.3s ease;
  }
  
  .card:hover {
    transform: translateY(-10px);
    box-shadow: 0 10px 30px rgba(0,0,0,0.2);
  }

EXPLICACIÓN:
  - transition: Define QUÉ propiedades se animan y en cuánto tiempo
  - transform: translateY(-10px): Mueve la card 10px hacia arriba
  - box-shadow: Crea una sombra más grande
  - ease: Tipo de animación (suave al inicio y al final)

TAMBIÉN SE AGREGÓ:
  Efecto en botones "Añadir al Carrito" con icono:
  <i class="fas fa-cart-plus"></i> Añadir al Carrito

8. MEJORA: Indicador de página activa en navbar
-----------------------------------------------------------
¿POR QUÉ ES IMPORTANTE?
  El usuario debe saber en qué página está navegando sin tener que adivinarlo.

SOLUCIÓN:
  Agregar la clase "active" al enlace de la página actual en el navbar.

EJEMPLO EN catalogo.html:
  <li class="nav-item">
    <a class="nav-link active" href="catalogo.html">Catálogo</a>
  </li>

ESTILO VISUAL:
  El enlace activo se verá con un color diferente y/o subrayado,
  indicando claramente al usuario dónde está.

CSS AGREGADO:
  .nav-link.active {
    font-weight: bold;
    border-bottom: 2px solid #0d6efd;
  }

CADA PÁGINA TIENE SU PROPIO ENLACE ACTIVO:
  - index.html: Ninguno (es la home)
  - catalogo.html: "Catálogo" activo
  - contacto.html: "Contacto" activo
  - retros.html: "Catálogo" activo (porque es una subcategoría)

-----------------------------------------------------------
9. MEJORA: Favicon agregado
-----------------------------------------------------------
¿QUÉ ES UN FAVICON?
  Es el pequeño icono que aparece en la pestaña del navegador,
  junto al título de la página.

VENTAJAS:
  - Profesionalidad
  - Fácil identificación cuando tienes múltiples pestañas abiertas
  - Branding de tu marca

CÓDIGO AGREGADO (en el <head> de cada página):
  <link rel="icon" type="image/x-icon" href="data:image/svg+xml,<svg xmlns='http://www.w3.org/2000/svg' viewBox='0 0 100 100'><text y='.9em' font-size='90'>⚽</text></svg>">

EXPLICACIÓN:
  - Usa un emoji ⚽ como favicon temporal
  - Formato SVG (escalable, se ve bien en cualquier tamaño)
  - data:image permite incrustar el SVG directamente sin archivo externo

PARA EL FUTURO:
  Puedes crear un favicon.ico profesional en sitios como:
  - favicon.io
  - canva.com
  - Adobe Express
