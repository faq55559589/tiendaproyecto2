═════════════════════════════════════════════════════════════════
  🚀 CÓMO USAR EL PROYECTO
═════════════════════════════════════════════════════════════════

1️⃣ BACKEND (Servidor API)

   Navegar a la carpeta:
   cd backend

   Instalar dependencias:
   npm install

   Configurar .env:
   (Crea un archivo .env con tus secretos como JWT_SECRET)

   Iniciar servidor:
   npm start

   El servidor correrá en: http://localhost:3000


2️⃣ FRONTEND (Aplicación Web)

   Opción A - VS Code Live Server:
   • Abre index.html
   • Click derecho → "Open with Live Server"

   Opción B - HTTP Server:
   npm install -g http-server
   http-server -p 8080

   El frontend correrá en: http://localhost:8080


3️⃣ BASE DE DATOS

   Base de datos SQLite:
   Ya viene pre-configurada en backend/database/golazostore.db.
   Para reiniciar, ejecuta: node backend/scripts/init-db.js
