## Alcance validado

- Flujo principal:
  - `home -> catalogo -> producto -> carrito -> checkout -> confirmacion -> mis-pedidos`
- Casos de error:
  - usuario no logueado en rutas privadas,
  - token invalido/expirado,
  - carrito vacio,
  - producto sin stock.
- Panel admin:
  - alta, edicion y baja de productos,
