  5. 💅 NUEVOS ESTILOS CSS
═════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│ ESTILOS AGREGADOS EN style.css                              │
└─────────────────────────────────────────────────────────────┘

  1. HERO SECTION (.hero-section)
     • min-height: 90vh
     • background: degradado negro/rojo
     • display: flex; align-items: center
     • overflow: hidden
     
     Pseudo-elementos:
       ::before - Patrón diagonal
       .hero-overlay - Gradiente radial rojo

  2. LOGO HERO (.hero-logo)
     • filter: drop-shadow(0 0 30px rgba(220,53,69,0.5))
     
  3. ANIMACIÓN PULSE (@keyframes pulse-animation)
     • 0%, 100%: scale(1)
     • 50%: scale(1.05)
     • Duración: 3s
     • Infinite loop

  4. ANIMACIÓN FADE-IN (@keyframes animate-fade-in)
     • from: opacity 0, translateY(30px)
     • to: opacity 1, translateY(0)
     • Duración: 1s

  5. FEATURE BOXES (.feature-box)
     • padding: 2rem 1rem
     • hover: translateY(-10px)
     • Icono hover: scale(1.2)

  6. NAVBAR OSCURO (.navbar-dark)
     • Links: rgba(255,255,255,0.85)
     • Hover: #dc3545
     • Brand hover: scale(1.05)

  7. BOTONES DANGER (.btn-danger)
     • background: #dc3545
     • hover: #bb2d3b
     • hover: translateY(-2px)
     • box-shadow rojo al hover

  8. CARDS MEJORADOS (.card)
     • border: 1px solid #f0f0f0
     • hover border: rgba(220,53,69,0.3)

  9. BOTÓN VOLVER ARRIBA (#backToTop)
     • background: #dc3545 (antes #0d6efd)
     • hover: #bb2d3b
     • box-shadow: rgba(220,53,69,0.4)

  10. BREADCRUMBS (.breadcrumb-item a)
      • color: #dc3545 (antes #0d6efd)
      • hover: #bb2d3b

  11. NAV ACTIVO (.nav-link.active)
      • border-bottom: 2px solid #dc3545
      • color: #dc3545

     • Estrellas de valoración
     • Fotos de clientes con productos

     • Integración con feed
     • Mostrar últimas fotos
     • Link directo

     • Crear favicon.ico con múltiples tamaños
